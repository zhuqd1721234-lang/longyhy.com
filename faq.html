// 选型相关
TGIC和HAA固化剂应该怎么选？▾
出口欧盟市场只能选HAA，因为TGIC已被列为SVHC高关注物质，不可出口欧盟。出口印度、土耳其、中东、东南亚等非欧盟市场，两套体系均可选用：TGIC户外耐候性能略优、价格较高（约$4,500–7,000/吨）；HAA完全合规、成本略低（约$3,500–5,500/吨）。不确定的客户建议两套体系都申请样品，按实际固化测试效果决定，详见固化剂详情页。
低温固化型和标准型树脂怎么选？▾
如果产线烘烤炉能耗成本较高，或基材对高温敏感，建议选低温固化型（150–160℃）；通用户外铝型材、家具五金场景用标准型（180–200℃）即可。把您的固化炉工艺参数发给我们，可免费获得选型建议。
// 采购与样品
五个标准件的最低起订量分别是多少？▾
聚酯树脂：样品级500kg起，正式采购3吨起
TGIC/HAA固化剂：样品200kg起，正式采购1吨起
流平剂+脱气剂：样品50kg起，正式采购200kg起
钛白粉：样品1吨起，正式采购3吨起
硫酸钡：样品1吨起，正式采购5吨起
具体规格与价格区间见五个标准件详情页。
申请样品后多久能收到？▾
标准情况下样品在5个工作日内寄出，具体到货时间取决于目的地清关效率。配方包整体询价的五个标准件会同批寄出，方便您统一测试周期。
配方包整体询价和单品类询价有什么区别？▾
配方包整体询价适合需要一次性配齐五个标准件的客户，我们按您提供的配方比例统一报价、同批寄样、出具整体配方兼容性验证报告。单品类询价适合已有部分供应商、只缺某一两个标准件的客户，可以单独询价，不强制整包采购。两个路径都可以在首页询价表单中选择。
// 付款与物流
支持哪些付款方式？▾
支持电汇（T/T）、信用证（L/C），以及通过第三方贸易平台的托管支付（信保订单）。首次合作客户可协商预付款+发货前尾款+验收后质保金的分批付款方式。具体条款见条款与隐私政策。
订单量不够一整柜怎么办？▾
配方包五个标准件支持同柜拼配出口，我方统一采购、统一入库、拼柜发出，运费和清关成本比分开采购更低，不额外收取拼柜手续费，详见条款说明。
// 质量与风险
如果货物到港后检测不合格怎么办？▾
正式合同中会写明质量赔偿条款：货到港后30天内，经第三方检测机构复检确认不符合合同约定规格的，我方承担全额退换或折价补偿，详见质量赔偿条款。
原料成本占粉末涂料总成本的比例是多少？▾
根据行业普遍情况，原料成本约占粉末涂料总成本的80%–90%，因此原料选型和价格稳定性对整体利润率影响很大——这也是我们提供配方风险核查服务和价格监控订阅的原因。
